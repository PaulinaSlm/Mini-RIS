# Mini-RIS 
Projekt na wzór szpitalnych systemów zarządzania danymi pacjentów i ich badaniami w zakładach radiologii

## Główne funkcje:
* Dodawanie nowych badań dla istniejących pacjentów
* Wprowadzanie nowego pacjenta do bazy
* Parametryzowanie badań w zależności od modalności (np. dawka promieniowania dla RTG lub wartość pola magnetycznego dla MRI) dzięki użyciu klas i dziedziczenia
* Usuwanie badań/pacjentów z bazy
* Zmiana danych badań/pacjentów
* Wyświetlenie zawartości bazy
* Generowanie raportów
* Trwałość danych, dzięki ich zapisywaniu do pliku CSV

## Użyte technologie:
* Python 3 (w tym OOP)
* Pandas, OS, datetime
* Praca z plikami
